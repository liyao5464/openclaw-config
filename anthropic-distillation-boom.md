# Anthropic 急了？一条推文暴露的 AI 霸权焦虑

昨天半夜，AI 圈炸了。

Anthropic 官方账号突然发了一条推文，措辞之严厉，在整个 AI 行业都极为罕见——直接点名三家中国公司：DeepSeek、Moonshot AI、MiniMax，指控它们对 Claude 发起了"工业级蒸馏攻击"。

2.4 万个假账号，1600 万轮对话，"窃取"Claude 的能力。

听起来很严重对吧？

但接下来发生的事，恐怕 Anthropic 自己都没想到。

---

## 排名里的小心思

先说一个大多数人没注意到的细节。

Anthropic 在推文里列出三家公司的顺序是：DeepSeek、Moonshot AI、MiniMax。

但实际调用数据呢？

DeepSeek 只调用了 **15 万次**。Moonshot AI 调用了 **340 万次**。MiniMax 调用了 **1300 万次**。

调用量最少的排第一，最多的排最后。

**这不是按调用量排序，这是按威胁程度排序。**

Anthropic 最忌惮的那个名字，被放在了最前面。DeepSeek V2 的性价比有多恐怖，业内人都清楚。15 万次调用就让 Anthropic 如此紧张——这恰恰说明，DeepSeek 的效率高到让人害怕。

---

## 马斯克一句话，把场子炸了

推文发出几小时后，马斯克来了。

他只回了一个词：**"Banger"**。

翻译成中文就是：绝了，经典。

但他不是在夸 Anthropic。紧接着他又补了一句：

> **"How dare they steal the stuff Anthropic stole from human coders??"**

他们怎么敢偷 Anthropic 从人类程序员那里偷来的东西？？

一句话，把 Anthropic 的底裤扒了个干净。

你想想：Anthropic 训练 Claude 用的是什么？是全网爬来的代码、文章、书籍。这些数据的原作者同意了吗？没有。付钱了吗？没有。

现在别人用你的 API 输出来训练模型，你就急了？

**这叫什么？这叫贼喊捉贼。**

---

## 1600 万轮，真的很多吗？

Anthropic 在推文里反复强调"1600 万轮对话"，想营造一种"规模巨大、触目惊心"的感觉。

但说实话，这个数字放在行业里，真不算什么。

一个中等规模的中文 AI 应用，日活用户每天就能产生几百万轮对话。国内随便一个 API 中转站，月调用量都不止这个数。

更关键的是——**这些账号是付费的**。

正常注册，正常充值，正常调用 API。Anthropic 收了钱，转头说人家"欺诈"？

这就好比开了家饭店，客人点菜付钱吃饭，你突然冲出来说："你们吃太多了！你们在偷我的菜谱！"

**不是，你定价的时候没算过成本吗？**

---

## 评论区彻底翻车

Anthropic 大概以为，发出这条推文后，评论区会是一片"干得漂亮"。

结果呢？

**清一色的嘲讽。**

最高赞评论只有一句话，简单粗暴：

> "HOW IS MODEL DISTILLATION STEALING"
> 
> 模型蒸馏怎么就成偷窃了？

290 个赞，还在涨。

前 GitHub 员工直接实名开怼：

> "我在 GitHub 工作的时候，我们不得不让你们停止过度使用我们的 API，因为你们违反 ToS 爬 GitHub 仓库。你们道了歉，然后继续这么做。"

**当年你们怎么对 GitHub 的，今天别人就怎么对你们。**

风水轮流转，这话一点不假。

---

## 一个经济学家的精准拆解

评论区里有一段话，我觉得说得最到位：

> "你们用全世界的数据训练模型，却反对别人从你们的输出中学习。在经济学里，这叫**寻租行为**——不靠创新维持优势，靠设置壁垒保护既得利益。"

最后一句是金句：

> **"Innovation loves competition. Moats hate it."**
> 
> 创新热爱竞争，护城河憎恨竞争。

这句话精准概括了 Anthropic 的焦虑本质——**他们保护的不是创新，是租金。**

当技术壁垒被突破，当护城河开始干涸，他们选择的不是跑得更快，而是回头骂追上来的人"你偷了我的鞋"。

---

## Anthropic 真正害怕的是什么？

表面上是"蒸馏攻击"，但 Anthropic 真正害怕的，是一个他们不愿意说出口的事实：

**中国 AI 追得太快了。**

DeepSeek V2 的性价比震惊了整个行业。月之暗面的 Kimi 在长文本上打出了自己的特色。MiniMax 在多模态领域持续发力。

这三家有一个共同点：用更少的资源，做出了接近甚至某些方面超越 Claude 的效果。

有人在评论区问了一个灵魂问题：

> "如果别人能把 Claude 蒸馏成更小、更便宜、更好的模型，你们自己为什么做不到？"

**如果你真的技术领先，为什么别人蒸馏一下就能追上？**

答案只有一个：差距没有你吹的那么大。

---

## 为什么发推文，不上法庭？

最后说一个很多人忽略的问题。

如果 Anthropic 真的掌握了确凿证据，为什么不起诉？为什么选择在社交媒体上"喊冤"？

因为**上法庭需要证据，发推文只需要情绪**。

他们想要的不是法律正义，是舆论同情。他们想在全球范围内建立一个叙事："中国公司在偷美国的 AI 技术。"

但评论区的反应说明，这个叙事卖不动。

因为所有人都记得：**你们当年爬全网数据的时候，可没问过任何人的意见。**

---

## 写在最后

马斯克那句话，值得再读一遍：

> **"How dare they steal the stuff Anthropic stole from human coders??"**

AI 行业的道德高地，谁都想抢，谁也站不稳。

与其发推文抱怨"被偷"，不如想想怎么把模型做得更好。

**毕竟，真正强大的技术，从来不怕别人学。**

怕别人学的，说明也就那样。
