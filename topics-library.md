## 📝 写作模板

### 工具测评模板
```
1. 工具简介（一句话说明是什么）
2. 核心功能（3-5个主要功能）
3. 适合人群（谁最适合用）
4. 收费情况（免费/付费）
5. 优缺点总结
6. 使用建议
```

### 科普文章模板
```
1. 引入（生活中的例子）
2. 是什么（通俗解释）
3. 怎么用（实际场景）
4. 注意事项（风险/局限）
5. 总结
```
