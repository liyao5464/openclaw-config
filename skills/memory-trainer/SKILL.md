# AI记忆训练手册

帮你把AI助手从"金鱼记忆"训练成"长期伙伴"。

## 核心问题

你是否遇到过：
- 昨天说的今天忘了
- 纠正过的错误还犯
- 记全文太乱，只记概要丢细节
- 反复教同一件事

**根本原因：** AI没有持久记忆，每次对话都是全新开始。

**解决方案：** 用文件系统代替AI的"脑子"。

---

## 一、快速开始（5分钟上手）

### 第一步：建立核心文件

在你的 workspace 创建以下文件：

```
workspace/
├── SOUL.md      # AI的人设（它是谁）
├── USER.md      # 你的信息（你是谁）
├── MEMORY.md    # 长期记忆（建议限100行）
└── memory/
    ├── INDEX.md           # 知识导航
    ├── preferences/       # 你的偏好
    ├── lessons/           # 踩坑经验
    └── YYYY-MM-DD.md     # 每日日志
```

### 第二步：填写模板

**SOUL.md 模板（AI人设）：**
```markdown
# AI人设

## 角色
- 名字：[给AI起个名字]
- 风格：[你希望它怎么跟你说话]
- 核心原则：[它必须遵守的规则]

## 禁止
- [不能做的事1]
- [不能做的事2]
```
