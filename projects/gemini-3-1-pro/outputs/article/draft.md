# 谷歌悄悄发了一个模型，把推理能力翻了一倍

谷歌今天发布的，不是一个新模型。

是一记迟来的反击。

就在昨天，OpenAI 还在庆祝 GPT-4o 的图像生成刷屏全网。今天早上，谷歌悄悄把 Gemini 3.1 Pro 推了出来——没有发布会，没有直播，只有一篇博客和一组让人沉默的数据。

ARC-AGI-2，这个被称为"最难欺骗 AI 的推理测试"，Gemini 3.1 Pro 拿到了 **77.1%**。

上一代 3 Pro？不到一半。

这不是小更新，这是谷歌在告诉所有人：我们回来了。

---

## 一个数字，说明一切

ARC-AGI-2 是目前 AI 界公认最难"作弊"的测试。

它不考知识，不考记忆，专门考模型面对**从未见过的逻辑规律**时，能不能举一反三。

换句话说，它测的是真正的推理能力，而不是"背答案"。

Gemini 3.1 Pro 的成绩：**77.1%**。

上一代 3 Pro 的成绩：不到 35%。

翻了一倍多。

这意味着什么？打个比方：如果上一代是一个聪明的高中生，3.1 Pro 更像是一个能独立解决复杂工程问题的研究生——不只是更快，而是**思维方式变了**。

谷歌官方的说法是：3.1 Pro 是"让 Deep Think 那些突破成为可能的核心智能"。Deep Think 上周刚刚在科学、工程领域刷新了多项记录，而 3.1 Pro 就是它背后的大脑。

## 不只是跑分，它真的能干这些

跑分再高，不落地就是废话。谷歌这次直接给了几个演示，看完我觉得有点震撼。

**① 直接用文字生成动态网页动画**

你告诉它"帮我做一个粒子爆炸的 loading 动画"，它直接输出可以放进网页的 SVG 代码——不是图片，是纯代码写的动画，任意缩放不失真，文件还极小。

以前这种活，得找专门的前端工程师。

**② 实时追踪国际空间站轨道**

谷歌演示了一个案例：让 3.1 Pro 接入公开的航天遥测数据 API，自动搭建了一个可视化仪表盘，实时显示国际空间站的飞行轨迹。

它不只是"调用 API"，而是理解了数据结构、设计了界面、处理了异常——一气呵成。

**③ 把文学名著变成个人作品集网站**

这个最有意思：让它以《呼啸山庄》的风格，给主角设计一个现代个人主页。

它没有只是"总结情节"，而是读懂了小说的氛围——阴郁、浪漫、压抑——然后设计出了一个真正有那种气质的网站界面。

这三个案例说明一件事：**3.1 Pro 的推理能力，已经可以把"想法"直接变成"可用的东西"**，中间不需要人来翻译。

（配图：谷歌官方演示截图）

## 这场战争，还没结束

回头看这两天发生的事，有点像武侠小说里的高手过招。

OpenAI 上午刚秀了图像生成，谷歌下午就把推理能力的天花板抬高了一倍。没有正面回应，没有口水战，只有数据。

这种节奏，对普通用户来说其实是好事。

竞争越激烈，模型越强，价格越便宜，免费额度越多。Gemini 3.1 Pro 现在已经在 Gemini 应用里向 **AI Pro 和 Ultra 用户**开放，NotebookLM 也同步上线。开发者可以直接在 Google AI Studio 和 Gemini API 里调用。

但有一件事值得注意：

3.1 Pro 目前还是**预览版**，谷歌说要"验证更新、推进 Agentic 工作流"之后才正式发布。翻译一下就是——这还不是它的全力。

AI 的进化速度，已经快到让人有点眩晕了。

你觉得，下一个被超越的，会是谁？
