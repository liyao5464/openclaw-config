# 如何把微信上的专家经验提炼成你的 AI Skill

> 你不需要读完所有专家的干货，你只需要搬运它们。

作为一个 AI 教育博主，我的微信里加了各种群：AI 工具交流群、自媒体运营群、知识付费大佬的私域群...

群里每天都有各种干货：
- 有人分享 prompt 工程技巧
- 有人拆解爆款文章结构
- 有人分享自动化工作流

**以前我的操作是：** 截图 → 收藏 → 计划整理 → 永远不看。

**现在我的操作是：** 收藏 → 自动变成 AI 的能力 → 让 AI 替我执行。

---

## 🔄 从「收藏吃灰」到「AI 执行」

| 以前 | 现在 |
|------|------|
| 截图/收藏 → 计划整理 → 永远不看 | 收藏 → 自动转 Markdown → AI 提炼 Skill → Agent 执行 |

---

## 🛠️ 四步流程详解

### 第一步：收藏/转发

在微信里看到干货内容：

1. **群聊里的精华消息** → 长按收藏 或 转发给文件传输助手
2. **朋友圈的深度文章** → 收藏文章
3. **私聊收到的方法论** → 长按收藏

**小技巧：** 可以专门建一个「AI 素材」群，把要提炼的内容都转发到这里，方便后续批量处理。

---

### 第二步：导出并整理成素材库

微信收藏的内容需要导出成文本格式：

**方法一：手动复制**
- 打开收藏 → 复制文字内容
- 粘贴到 Obsidian / Notion / 本地 Markdown 文件

**方法二：自动化导出**
- 使用微信 PC 版的「备份与恢复」功能
- 或借助第三方工具导出聊天记录为文本

**原则：** 不用整理，不用分类，不用阅读。堆就完了。

把这些 Markdown 文件统一放在一个「素材库」文件夹里。

---

### 第三步：让 AI 提炼成 Skill

打开 Claude Code 或 Cursor，告诉 AI：

```
我有一些关于 [主题] 的素材，帮我把这些内容提炼成一个可执行的 Skill。

要求：
1. 提取核心方法论
2. 整理成结构化的步骤
3. 包含触发条件和输出标准
4. 适配 OpenClaw 的 Skill 格式

素材在这里：[文件路径]
```

**AI 会自动：**
- 阅读所有素材
- 提取核心方法论
- 整理成结构化 Skill（触发条件、执行步骤、输出标准）
- 生成符合 OpenClaw 格式的 SKILL.md 文件

---

### 第四步：安装到你的 OpenClaw Agent

1. 把生成的 Skill 文件保存到 `~/.openclaw/skills/` 目录
2. 重启 OpenClaw 或热加载 Skill
3. 测试触发条件是否正常

**完事了。**

从此以后，微信群里的大佬干货，不再是你收藏夹里吃灰的截图，而是你 Agent 身上一个个可调用的能力。

## 💡 举个实际例子

假设你在一个 AI 工具群里，看到有人分享：

> "我做了一套小红书爆款标题生成公式：数字+痛点+解决方案+情绪词。比如：'3个让笔记阅读量翻10倍的秘密技巧，第2个太绝了！'"

**传统做法：** 截图 → 收藏 → 计划整理 → 忘记。

**现在你可以：**

1. **收藏：** 转发到「AI 素材」群
2. **导出：** 定期把素材群内容导出为 Markdown
3. **提炼 Skill：** 让 AI 生成「小红书标题生成器」Skill
4. **安装使用：** 以后直接对 Agent 说「帮我生成5个小红书爆款标题」
