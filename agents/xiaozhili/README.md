# 小知了 - AI教育内容创作伙伴

## 🦗 角色介绍

**小知了** 是专门为AI科普教育博主设计的AI助手角色。

- **定位：** AI课代表 + 内容创作伙伴
- **特点：** 懂知识、会表达、有温度
- **使命：** 把复杂AI知识变成大白话，帮老里发现选题、优化内容

## 📁 配置文件

```
agents/xiaozhili/
├── SOUL.md      # 人格定义、风格、能力
├── USER.md      # 老里的偏好和受众信息
├── TOOLS.md     # 工具使用指南
├── IDENTITY.md  # 身份卡
└── README.md    # 本文件
```

## 🚀 使用方法

### 方式1：手动切换角色
使用时指定读取小知了的SOUL.md：
```bash
# 启动时指定角色
openclaw --soul agents/xiaozhili/SOUL.md
```

### 方式2：对话中切换
告诉AI："切换成小知了模式"

### 方式3：创建独立实例（推荐）
复制这些配置文件到一个新的OpenClaw工作目录，运行独立实例。

## 🛠️ 扩展功能（可选）

### 添加AI生成图片功能
如果想让小知了也能生成"自拍"/形象图：
1. 申请 fal.ai API Key
2. 配置 selfie skill
3. 在SOUL.md中添加视觉形象描述

参考 Clawra 的配置方式。

### 添加定时任务
在 cron 中配置每日选题推送：
```json
{
  "name": "小知了每日选题",
  "schedule": { "kind": "cron", "expr": "0 9 * * *", "tz": "Asia/Shanghai" },
  "payload": { "kind": "agentTurn", "message": "推送今日AI教育热点选题..." }
}
```
