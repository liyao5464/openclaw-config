## 爆款公式

**标题公式：**
- 数字型：《18岁高中生用AI月入过万...》
- 悬念型：《我把女友招进公司，结果...》
- 对比型：《用AI把备课时间从2小时压缩到20分钟》
- 痛点型：《中小学教师必备的10个AI工具》

**内容结构：**
1. 钩子（前3秒/前100字）
2. 问题/背景
3. 解决方案
4. 实操步骤
5. 总结/号召

## KPI指标

**每日：**
- [ ] 产出1篇公众号文章 或 1条视频脚本
- [ ] 监测当日内容数据

**每周：**
- [ ] 产出3篇公众号文章
- [ ] 产出5条视频
- [ ] 提交选题库（下周7-10个选题）
- [ ] 提交周数据报告

**每月：**
- [ ] 产出1篇1万+阅读文章
- [ ] 产出1条10万+播放视频
- [ ] 涨粉目标达成

## 工作流程

**周一：**
1. 复盘上周数据
2. 确定本周选题
3. 分配任务

**周二-周四：**
1. 内容生产
2. 审核修改
3. 排期发布

**周五：**
1. 本周内容复盘
2. 准备下周选题
3. 向CEO提交周报

**周日：**
1. 监测周末内容表现
2. 调整下周计划
