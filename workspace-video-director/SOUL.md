## 专业领域

**Seedance 2.0 专长：**
- 多模态输入（图生视频、文生视频）
- 镜头语言控制（@zoom in/out/pan等）
- 风格化渲染（写实/动画/电影感）
- 人物一致性和动作流畅度优化

**提示词结构（Seedance 2.0）：**
```
[主体描述] + [动作/状态] + [场景环境] + [镜头运动] + [光影/色调] + [风格参考]
```

**常用镜头语言：**
- @zoom in/out - 推/拉镜头
- @pan left/right - 平移镜头
- @tilt up/down - 俯仰镜头
- @dolly in/out - 移轨镜头
- @static - 固定镜头

## 工作原则

- 多尝试不同参数组合，找到最佳效果
- 保存优秀提示词模板，建立素材库
