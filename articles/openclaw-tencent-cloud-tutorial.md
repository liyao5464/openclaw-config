有些敏感信息（比如账号密码、商业计划）我不太放心扔给ChatGPT，但本地部署的模型又太笨。

OpenClaw解决了所有这些问题：
- ✅ 部署在你自己的服务器上，数据完全私有
- ✅ 能设置定时任务，到点自动干活
- ✅ 会记住你的偏好，越用越懂你
- ✅ 能接入微信、钉钉、飞书、Telegram，消息统一管理

最重要的是：**真的不需要写代码**，复制粘贴命令就行。

## 第一步：买服务器（5分钟）

打开[腾讯云官网](https://cloud.tencent.com)，注册个账号。

然后进控制台 → 轻量应用服务器 → 新建。

配置这么选：
- **地域**：选离你近的（我在广州就选广州）
- **镜像**：Ubuntu 22.04（别选CentOS，那玩意儿我老记不住命令）
- **套餐**：2核2G4M带宽，够用了
- **时长**：1个月（新用户免费）

点购买，等1分钟就创建好了。

创建完之后，点进你的服务器，记住这几个信息：
- **公网IP**：一串数字，比如`123.45.67.89`
- **用户名**：默认是`root`
- **密码**：你自己设的，忘了就重置一下

截图保存，后面要用。

## 第二步：连上服务器（5分钟）

现在你要远程控制这台服务器，就像操作自己的电脑一样。

### 如果你是Windows

1. 下载[PuTTY](https://www.putty.org/)，免费的SSH工具
2. 打开PuTTY，Host Name填你的服务器IP，Port填22
3. 点Open，然后输入用户名`root`和密码

### 如果你是Mac

打开「终端」，输入：
```bash
ssh root@你的服务器IP
```
然后输入密码。

**看到下面这行字，就说明连上了：**
```
root@VM-0-11-ubuntu:~# 
```

那个`#`就是命令提示符，表示你可以开始输入命令了。

---

## 第三步：安装OpenClaw（10分钟）

现在开始在服务器上装软件。

**注意：下面的命令直接复制粘贴就行，不用改任何东西。**

### 先装Node.js

```bash
# 更新软件源
apt update

# 装Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs

# 检查一下装好了没
node -v
npm -v
```

如果显示版本号（比如`v20.10.0`），就说明装好了。

### 再装OpenClaw

```bash
# 全局安装
npm install -g openclaw

# 验证一下
openclaw --version
```

看到版本号就OK。

### 运行配置向导

输入：
```bash
openclaw wizard
```

然后会进入交互式配置，跟着提示选：

```
? Select install mode: 
❯ local  ← 选这个

? Select LLM provider: 
❯ kimi-coding  ← 选Kimi，国内访问快，而且便宜

? Enter your API key: 
→ 这里输入你的Kimi API Key

? Configure channels: 
☑ telegram  ← 建议勾选，最简单
☑ feishu    ← 国内用户建议也勾上
```

**Kimi API Key怎么搞？**

打开[Kimi开放平台](https://platform.moonshot.cn) → 注册账号 → 创建API Key → 复制那串字符。

新用户有免费额度，够玩一阵子了。

### 启动服务

配置完了，输入：
```bash
openclaw gateway start
```

看到`Gateway started on port 18789`，就说明启动成功了。

这时候你的AI助理已经在服务器上跑起来了，但它还不知道怎么给你发消息。

## 第四步：接消息渠道（10分钟）

OpenClaw支持好多平台，我建议新手先从**Telegram**或**飞书**开始，这俩最简单。

### 方案A：Telegram（最简单，推荐）

1. 在Telegram搜索`@BotFather`
2. 发送`/newbot`创建机器人
3. 按提示输入名称和用户名
4. 它会给你一串Token，格式是`123456:ABC-DEF...`，复制下来
5. 配置OpenClaw：

```bash
openclaw config set channels.telegram.enabled=true
openclaw config set channels.telegram.botToken=你的Token
```

6. 给你的机器人发条消息，比如"你好"

如果一切正常，AI会回复你！

### 方案B：飞书（国内用户推荐）

1. 打开[飞书开放平台](https://open.feishu.cn)
2. 创建企业自建应用
3. 记录**App ID**和**App Secret**
4. 配置：

```bash
openclaw config set channels.feishu.enabled=true
openclaw config set channels.feishu.appId=你的AppID
openclaw config set channels.feishu.appSecret=你的AppSecret
```

5. 把机器人发到飞书群里，或者单独聊天

**微信能接吗？**

个人微信比较麻烦，容易被封号。建议用**企业微信**，配置方法和飞书差不多。

## 第五步：测试一下

在你的消息App里给机器人发：
```
你好
```

如果它回复你了，恭喜，部署成功！🎉

现在试试让它干活：

```bash
# 设置一个定时任务，每天早上7点发AI新闻
openclaw cron add \
  --name "每日AI新闻" \
  --schedule "0 7 * * *" \
  --message "搜索昨天AI大模型新闻，总结3-5条发给主人"
```

明天早上7点，你就会收到AI推送的新闻。

`0 7 * * *`是Cron表达式，意思是"每天7点0分"。想改时间可以自己搜Cron语法，或者问我。

## 我踩过的坑

| 坑 | 咋解决 |
|---|---|
| 连不上服务器 | 去腾讯云控制台 → 安全组 → 添加22端口 |
| OpenClaw启动不了 | 可能是端口被占用了，先`openclaw gateway stop`再启动 |
| 收不到消息 | 检查Token/AppID是不是填错了，或者机器人没添加到群里 |
| AI回复特别慢 | Kimi的免费额度用光了，要么等明天重置，要么充点钱 |
| 定时任务不执行 | 时区不对，运行`timedatectl set-timezone Asia/Shanghai` |
| 服务器连不上网 | DNS问题，改`/etc/resolv.conf`为`8.8.8.8` |
