所以今天不写别的，就教你怎么给OpenClaw做备份。

其实特别简单。两步就够。

**第一步，搞个GitHub的PAT（Personal Access Token）。**

浏览器打开github.com，右上角点你头像，Settings → 拉到底 Developer settings → Personal access tokens → Tokens (classic)。

新建一个，Note随便填，比如"Openclaw备份"。

Expiration选90天，别选永久。

Scopes（权限）那里，**只勾repo**就行，别的不用管。

生成之后，**立刻复制**。这玩意儿只显示一次，回头再看就没有了。

复制完干嘛？去GitHub新建一个私有仓库，获取仓库地址。类似这样：https://github.com/你用户名/你的仓库名

**第二步，装gitclaw。**

回到OpenClaw，直接发这句话给它：

> 安装gitclaw用于在我的私有github库地址： https://github.com/你用户名/你的仓库名 Use this PAT: 你的PAT token 创建 .gitignore 排除 .env, tokens等敏感文件

然后确认一下，它会说"gitclaw已安装"。

搞定。

以后你所有的对话记录、配置文件、Agent设置，都会**自动备份**到这个私有仓库。

---

如果真的不幸崩了，怎么办？

更简单。

重新装好OpenClaw之后，一行命令：

```
git clone https://github.com/你用户名/你的仓库名 ~/.openclaw/workspace
```

全部回来。

就这么快。
