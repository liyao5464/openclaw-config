## 一、先说说它到底是个啥

你可能用过 ChatGPT，用过 Claude，用过 DeepSeek。

但你有没有发现一个问题——

**你得主动去找它。**

你打开网页，输入问题，等它回答，关掉网页。

它永远在那里等你，从来不会主动找你。

就像一个员工，你不叫他，他就坐在工位上发呆。

OpenClaw 不一样。

它是一个跑在你服务器上的 AI 助理框架，可以接入 Telegram、微信、飞书等各种聊天软件。

它有定时任务，有主动推送，有长期记忆。

你睡觉的时候，它在帮你盯着新闻。

你起床的时候，它已经把今天的日程、热点、待办事项整理好发给你了。

你出门在外，随手发一条消息，它就开始在后台帮你干活。

**这才是我理想中的 AI 助理。**

不是一个等你来问的工具，而是一个真正在帮你跑的员工。

## 二、你需要准备什么

说实话，这玩意儿的门槛，比我想象的要低很多。

你需要的东西就三样：

**1. 一台服务器（或者一台不关机的电脑）**

推荐买一台云服务器，腾讯云、阿里云都行，最便宜的那种就够用了，一个月几十块钱。

如果你有一台家里不用的旧电脑，也可以，只要它能联网、能开机就行。

**2. 一个 Telegram 账号**

国内用户可能需要科学上网，但这个应该难不倒你。

Telegram 是目前接入最稳定的渠道，强烈推荐。

**3. 一个 AI 模型的 API Key**

OpenClaw 本身不带模型，你需要自己接一个。

推荐新手用 MiniMax 或者 DeepSeek，便宜，效果也不错。

就这三样，没了。

## 三、开始安装

好，废话不多说，直接开始。

**第一步：连上你的服务器**

如果你用的是云服务器，打开终端（Mac 用自带的 Terminal，Windows 用 PowerShell），输入：

```bash
ssh root@你的服务器IP
```

输入密码，进去。

如果你是第一次用服务器，看到一堆英文不要慌，这很正常。

**第二步：安装 OpenClaw**

一行命令搞定：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

等它跑完，你会看到安装成功的提示。

就这么简单，真的，有手就行。

**第三步：初始化配置**

运行：

```bash
openclaw onboard
```

它会问你几个问题：
- 用什么模型？（选 MiniMax 或者 DeepSeek，输入你的 API Key）
- 用什么渠道？（选 Telegram）
- Telegram Bot Token 是什么？

最后一个问题需要你先去创建一个 Telegram Bot。

**第四步：创建你的 Telegram Bot**

打开 Telegram，搜索 `@BotFather`，发送 `/newbot`。

它会让你给 Bot 起名字，随便起，比如"我的AI助理"。

然后它会给你一串 Token，复制下来，粘贴到刚才的配置里。

**第五步：启动！**

```bash
openclaw gateway start
```

然后去 Telegram 找到你刚创建的 Bot，发一条消息。

如果它回复你了——

**恭喜，你的 AI 助理上线了。**

## 四、让它真正"活"起来

装好只是第一步。

真正让它有用的，是配置它的"灵魂"。

OpenClaw 有一个工作区（workspace），里面有几个关键文件：

**SOUL.md** — 它是谁，什么性格，怎么跟你说话

**USER.md** — 你是谁，你的习惯，你的偏好

**HEARTBEAT.md** — 它每隔一段时间要主动检查什么

你可以把这些文件想象成"入职培训材料"。

你写得越详细，它就越懂你。

比如我的 SOUL.md 里写了：

> 你是老里的私人助理，说话要口语化，不要太正式，偶尔可以开个玩笑。

然后它就真的开始说人话了，不再是那种"您好，请问有什么可以帮助您的"的客服腔。

**定时任务**也是精髓所在。

你可以设置：
- 每天早上8点，推送今日新闻和日程
- 每天晚上10点，发一条晚安问候
- 每周一，总结上周的工作进度

这些都是几行配置的事，但配好之后，你会发现每天醒来手机上已经有一份"早报"在等你。

那种感觉，真的很上头。

## 五、常见问题

**Q：我不会用命令行怎么办？**

说实话，你只需要会复制粘贴就够了。

上面那几行命令，照着敲就行，出错了把报错信息发给 AI，让它帮你解决。

**Q：服务器要多少钱？**

腾讯云轻量应用服务器，最便宜的配置，一个月大概 40-60 块钱。

学生有优惠，更便宜。

**Q：API Key 怎么申请？**

去对应模型的官网注册账号，一般都有免费额度。

MiniMax 官网：platform.minimaxi.com
DeepSeek 官网：platform.deepseek.com

**Q：安全吗？会不会泄露我的数据？**

OpenClaw 是开源的，代码都在 GitHub 上，你可以自己看。

数据存在你自己的服务器上，不经过任何第三方。

这一点，比很多商业产品都要安全。
