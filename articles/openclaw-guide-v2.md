## 准备工作：你需要这三样东西

在开始之前，先把需要的东西列清楚。

**第一样：一台服务器**

服务器就是一台24小时开机、联网的电脑，放在机房里，你通过网络远程控制它。

你不需要买实体机器，直接租云服务器就行。

推荐腾讯云轻量应用服务器，最便宜的配置够用了。

价格：大概40-60块钱一个月，学生有优惠，更便宜。

**第二样：一个Telegram账号**

Telegram是一个国外的聊天软件，OpenClaw通过它跟你通信。

国内用户需要科学上网才能用，这个我就不展开说了，相信你有办法。

**第三样：一个AI模型的API Key**

API Key就是一串密码，让OpenClaw能调用AI模型的能力。

推荐新手用MiniMax，注册就有免费额度，够用很久。

好，东西准备好了，开始。

## 第一步：买一台服务器

打开腾讯云官网：cloud.tencent.com

注册账号，实名认证（必须的，跳不过）。

然后点顶部菜单"产品" → "轻量应用服务器"。

进去之后，选配置：

- 地域：选离你近的，比如上海或广州
- 镜像：选"Ubuntu 22.04"（重要！不要选其他的）
- 套餐：选最便宜的那个，2核2G，够用了

付款，等它开机，大概1-2分钟。

开机之后，你会看到一个IP地址，比如 `123.456.789.0`。

**把这个IP地址记下来，后面要用。**

---

## 第二步：连上你的服务器

这一步很多人会卡住，但其实很简单。

**如果你用Mac：**

打开"终端"（在应用程序 → 实用工具里找，或者直接搜索"Terminal"）。

输入这行命令（把IP换成你自己的）：

```
ssh root@123.456.789.0
```

按回车，它会问你密码。

密码在腾讯云控制台里，找到你的服务器，点"重置密码"设置一个。

输入密码的时候屏幕上不会显示任何东西，这是正常的，盲打就行，打完按回车。

看到一堆英文，最后出现 `root@VM-xxx:~#` 这样的提示符，说明连上了。

**如果你用Windows：**

打开PowerShell（在开始菜单搜索"PowerShell"）。

同样输入：

```
ssh root@123.456.789.0
```

后面步骤一样。

**如果连接失败：**

检查IP地址有没有输错，检查服务器有没有开机，检查密码有没有输对。

实在不行，把报错信息截图，发给AI让它帮你解决。

---

## 第三步：安装OpenClaw

连上服务器之后，复制粘贴这一行命令：

```
curl -fsSL https://openclaw.ai/install.sh | bash
```

按回车，等它跑完。

你会看到一堆安装进度，最后出现"Installation complete"或者类似的成功提示。

**就这一行，装好了。**

如果报错说"curl: command not found"，先运行这个：

```
apt-get install curl -y
```

然后再重新运行安装命令。

---

## 第四步：创建你的Telegram Bot

这一步需要你打开Telegram。

在搜索框里搜索 `@BotFather`，找到那个有蓝色对勾的官方账号，点进去。

发送：`/newbot`

它会问你：

**"Alright, a new bot. How are we going to call it? Please choose a name for your bot."**

这是让你给Bot起个名字，随便起，比如"我的AI助理"或者"老里的小助手"。

然后它会问：

**"Good. Now let's choose a username for your bot."**

这是Bot的用户名，必须以"bot"结尾，比如"laoli_assistant_bot"。

如果提示用户名已被占用，换一个就行。

成功之后，BotFather会发给你一串Token，长这样：

```
1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ
```

**把这串Token复制下来，非常重要，后面要用。**

---

## 第五步：申请AI模型的API Key

打开MiniMax开放平台：platform.minimaxi.com

注册账号，登录。

进去之后找"API Key"或者"密钥管理"，创建一个新的Key。

它会给你一串以"eyJ"开头的长字符串。

**复制下来，保存好，只显示一次。**

---

## 第六步：配置OpenClaw

回到你的服务器终端，运行：

```
openclaw onboard
```

它会问你几个问题，按照下面的回答：

**选择模型提供商：** 选MiniMax，然后粘贴你的API Key

**选择渠道：** 选Telegram

**输入Bot Token：** 粘贴刚才BotFather给你的Token

**你的Telegram用户ID：** 这个需要去Telegram搜索 `@userinfobot`，发任意消息，它会告诉你你的ID，是一串数字

全部填完，它会自动配置好。

---

## 第七步：启动！

运行：

```
openclaw gateway start
```

看到"Gateway started"或者类似提示，说明启动成功了。

现在去Telegram，找到你刚创建的Bot，发一条消息。

比如发"你好"。

**如果它回复你了——恭喜，你的AI助理上线了！**

---

## 第八步：让它真正懂你（最重要的一步）

很多人装好就完事了，然后发现它回复得很官方，跟普通AI没区别。

这是因为你还没有给它"入职培训"。

OpenClaw的工作区里有几个关键文件，你需要编辑它们。

**编辑SOUL.md（告诉它是谁）：**

在服务器上运行：

```
nano /root/.openclaw/workspace/SOUL.md
```

把里面的内容改成你想要的风格，比如：

```
你是我的私人助理，名字叫小助手。
说话要口语化，不要太正式。
我喜欢简洁直接，不喜欢废话。
偶尔可以开个玩笑，但不要太过分。
```

写完按 Ctrl+X，然后按Y，然后按回车保存。

**编辑USER.md（告诉它你是谁）：**

```
nano /root/.openclaw/workspace/USER.md
```

写上你的基本信息：

```
我叫[你的名字]，是[你的职业]。
我的时区是北京时间（GMT+8）。
我的作息大概是早9点到晚11点。
我喜欢[你的偏好]，不喜欢[你不喜欢的]。
```

**编辑HEARTBEAT.md（告诉它主动检查什么）：**

```
nano /root/.openclaw/workspace/HEARTBEAT.md
```

写上你想让它定期检查的事情：

```
每天早上检查：
- 今日日历事件
- 重要待办事项

如果有紧急的事，主动通知我。
```

**重启让配置生效：**

```
openclaw gateway restart
```

重启之后，再去Telegram发消息，你会发现它的回复风格完全不一样了。

## 常见问题

**Q：服务器断开连接了怎么办？**

重新运行 `ssh root@你的IP` 连上去，然后运行 `openclaw gateway start` 重新启动就行。

如果想让它开机自动启动，运行：

```
openclaw gateway enable
```

以后服务器重启，它会自动跑起来。

**Q：Bot没有回复我怎么办？**

先检查Gateway有没有在跑：

```
openclaw gateway status
```

如果显示stopped，运行 `openclaw gateway start` 重新启动。

**Q：回复很慢怎么办？**

正常现象，AI模型需要时间思考。

如果超过30秒没回复，可能是网络问题，重新发一次。

**Q：API Key用完了怎么办？**

去MiniMax平台充值，或者换一个模型的API Key。

DeepSeek也很便宜，可以备用。
