# 通用方法论

## L0: 老里风格的写作和AI使用方法论

## L1:
- 写作核心：真实故事 > 理论说教
- AI使用方法：让AI帮你干活，你来把控方向
- 记忆方法论：ePro-Memory六类分类+L0/L1/L2三层

## L2:
### 老里写作风格（style-agent）
- 核心定位：接地气的技术布道者
- 开头公式：大家好，我是老里，50-100字切入
- 情绪弧线：热点钩子→亲身遭遇→干货内容→哲学升华→行动号召
- 结尾公式：感恩三连（不可省略）
- 语言要求：用"你"不用"您"，短句一行，口语化
- 禁止：书面腔、AI味句式、说教口吻

### ePro-Memory方法论
- 六类分类：用户信息/偏好习惯/相关事物/发生的事/AI经验/通用方法
- 三层结构：L0摘要/L1概要/L2详情
- 检索逻辑：默认读L0，相关才读L1，要细节才读L2
- 分类内去重，避免记忆库臃肿

### 写作协作流程（强制）
1. 读style-agent.md（每次）
2. 读writing-guide.md（每次）
3. 问老里要真实素材
4. 写情绪大纲确认
5. 写初稿
6. 过审稿检查清单
7. 发老里审核
