# AI工作经验

## L0: 写公众号文章和配图的工作经验总结

## L1:
- 文章写作：按style-agent规范，先读文件再写
- 配图生成：用gen-image.sh，4张关键图
- 微信发布：用md2html-pro.js转HTML，推草稿

## L2:
### 文章工作流
1. 读style-agent.md + writing-guide.md（强制）
2. 构思情绪大纲，老里确认
3. 写初稿
4. 配图（关键节点4张）
5. 转HTML：node scripts/md2html-pro.js article.md wechat
6. 生封面：bash scripts/gen-cover-smart.sh "标题" cover.png
7. 推草稿：export WECHAT_APP_ID/SECRET && npx -y bun wechat-api.ts

### 配图经验
- 4张关键图：痛点/分类/层级/效果
- 用简洁英文提示词，避免复杂描述
- 风格：minimalist flat illustration, warm colors

### 发布检查清单
1. 确认发哪个号（主号/实验室）
2. 检查标题是否合适
3. 删掉备用标题
4. 封面提示词要简洁
5. 用md2html-pro.js统一转HTML
