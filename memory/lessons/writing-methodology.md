# 写作方法论

## 核心认知
AI最大的问题不是写得不好，是写得太"对"了。宁可乱一点，也不要完美。
没有真实素材就不动笔，编出来的一定是AI味。

## 结构公式
故事 → 冲突 → 框架 → 升华

## 三大根因（AI味）
1. 套用AI默认科普模板，不是老里的方法论
2. 编造体验，没有真实细节
3. 缺少情绪曲线

## 爆款标题要素
1. 流量关键词（热点/大名词/情绪词）
2. 冲突悬念（反常识/场景化/前后反差）
3. 情绪钩子（贪婪/恐惧/懒惰/痛点）
4. 数字
5. 动词

## 写作协作流程（强制）
1. 读 style-agent.md（每次）
2. 读 writing-guide.md（每次）
3. 问老里要真实素材
4. 写情绪大纲确认
5. 写初稿 → 过检查清单 → 老里审核
