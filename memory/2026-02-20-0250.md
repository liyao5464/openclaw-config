## 和竞品比，Gemini 3.1 Pro 站在哪里？

坦白说，AI 模型的"排行榜"每隔几周就会洗牌一次。但这次 Gemini 3.1 Pro 的数据，确实值得认真看一眼。

### 基准测试对比

| 测试项目 | Gemini 3.1 Pro | 说明 |
|---------|---------------|------|
| ARC-AGI-2 | **77.1%** | 举一反三能力，上代不到40% |
| GPQA Diamond | **94.1%** | 研究生级科学推理 |
| SciCode | **58.9%** | 科学计算Python编程 |
| Terminal-Bench Hard | **53.8%** | Agent自主编程能力 |
| HLE | **44.7%** | 接近人类极限的推理 |

### 三个维度看差异

**1. 推理能力：Gemini 领先**

ARC-AGI-2 这个测试，专门防止模型"背答案"。77.1% 的成绩，目前是公开榜单最高分之一。

**2. 价格：Gemini 最便宜**

同等性能下，Gemini 3.1 Pro 的价格大约是 Claude Opus 4.6 和 GPT-5.2 的一半。对于要大规模调用 API 的开发者来说，这个差距非常实际。

**3. 生态：Google 全家桶优势**

- Gemini app（消费者）
- NotebookLM（知识管理）
- Vertex AI（企业）
- Android Studio（移动开发）
- Google Antigravity（Agent 开发）

一个模型，打通了 Google 整个产品线。这是 OpenAI 和 Anthropic 目前还没有的优势。

### 但也有不足

目前 3.1 Pro 还是**预览版**，正式版还没发布。Google 说要先收集反馈，再推进 Agent 工作流的优化。

换句话说：**现在能用，但还不是最终形态。**
