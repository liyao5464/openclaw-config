## OpenClaw 使用心得（老里整理）

1. 遇到不会的，直接让 OpenClaw 自己来配置，比如安装 Skills
2. OpenClaw 的 Web 页面可以把语言设置为中文
3. 过程中一定要开科学上网
4. ClawHub 需要登录，否则会限制速率；安装 Skills 也必须登录
5. 安装 Twitter、YouTube 等 Skills 需要通过 ClawHub 登录后操作
6. OpenClaw 想做更多事就得给更多权限（登录账号、配置 API）；建议申请单独的 Google 账号给它用
