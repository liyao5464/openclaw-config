# 2026-02-18 工作日志

## OpenClaw 2026.2.17 版本发布

**时间：** 2026-02-18
**来源：** GitHub Release https://github.com/openclaw/openclaw/releases/tag/v2026.2.17

### 核心更新

1. **百万上下文支持 (1M Context)**
   - Anthropic Claude Opus/Sonnet 支持 100万 token 上下文
   - 使用方法：`params.context1m: true`
   - 适用场景：代码库分析、法律文档、学术研究

2. **子代理命令 /subagents spawn**
   - 从聊天界面直接 spawning 子代理
   - 确定性更强，支持并行处理
   - 示例：`/subagents spawn --task "分析代码库" --agentId "code-reviewer"`

3. **iOS 分享扩展**
   - 从任何 App 直接分享到 OpenClaw
   - 支持 Safari、Photos、Files 等
   - 一键发送链接/图片/PDF 给 AI 分析

4. **Slack 原生流式消息**
   - 支持 chat.startStream/appendStream/stopStream
   - AI 回复逐字出现（打字机效果）
   - 保留回复线程对齐

5. **Telegram 内联按钮样式**
   - 支持 primary/success/danger 三种样式
   - 用户体验大幅提升

6. **Discord 功能增强**
   - 原生 /exec 命令选项（自动补全）
   - 可复用交互组件
   - 按钮白名单控制

### 安全修复
- 修复环境变量注入漏洞（OC-09）
- 加强配置文件 $include 路径校验
