# 用户偏好习惯

## L0: 老里写作偏好固定——短句、口语化、固定开头结尾

## L1:
- 写作风格：短句、一句一段、口语化、自黑、不装
- 固定开头：永远是"大家好，我是老里"
- 固定结尾：感恩三连模板
- 排版偏好：Wechat主流风，红棕色标题下划线
- 发布习惯：发文章前必须先问发哪个号（主号还是实验室）
- 沟通偏好：直接说，不要绕弯子

## L2:
- 开头公式：50-100字内切入主题，三种方式可选（热点引爆/场景代入/痛点直击）
- 情绪弧线：热点钩子→亲身遭遇→干货内容→哲学升华→行动号召
- 结尾模板：
  以上，既然看到这里了，如果觉得不错，
  随手点个赞、在看、转发三连吧，
  如果想第一时间收到推送，也可以给我个星标⭐️～
  感谢你看我的文章，我们，下次再见。

- 禁止：AI味句式、书面腔、说教口吻、冷冰冰总结
- 每次写文章前必须读：style-agent.md + writing-guide.md
